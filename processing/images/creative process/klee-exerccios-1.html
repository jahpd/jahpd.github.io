Klee Exercícios (1)

Estava re-lendo a tese do Alex Mclean e resolvi subir três códigos onde ele exemplifica o processo criativa do improvisador-programador. O sketch foi feito com um plugin jekyll que fiz a uns dois anos.

abstratoum

function rx(){
  return random(width);
}
	
function ry(){
  return random(height);
}

createCanvas(480, 360);
background(255);
var i;
for(i=0; i < 20; i++) {
  bezier(rx(),ry(),rx(),ry(),rx(),ry(),rx(),ry());
}
