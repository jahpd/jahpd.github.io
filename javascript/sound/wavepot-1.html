Wavepot (1)

Acabei de atualizar alguns plugins para Jekyll. Dessa vez para síntese sonora.

É bastante simples; algumas funções são carregadas diretamente. Este post é pra testar uma senóide. Abaixo exemplifico como utilizar no Jekyll:

Primeiro é necessário chamar o script:

{% wavepotload %}

Depois escrevemos um código que rederiza:

{% wavepot quatrocentos %} sin(442,0.71) {% endwavepot %}
